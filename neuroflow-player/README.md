# NeuroFlow Player

An advanced mobile-first media player built with React, Web Audio API, and Framer Motion.

## Features
- 🎵 Audio & video playback with equalizer
- 🌊 5 real-time audio visualizers (Wave, Bars, Fluid, Circular, Particles)
- 🧠 Neuro-Sync biofeedback simulation
- 🎮 Gesture controls (swipe to skip/change visualizer)
- 📱 Motion control support
- 🌐 Immersive/social modes
- ⚙️ Advanced audio effects & spatial audio
- 📂 Folder scan & file picker for local media
- 🔁 Shuffle, repeat, playback speed
- 📲 PWA-ready with service worker

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo
4. Framework: **Create React App** (auto-detected)
5. Click **Deploy** — done!

## Local Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
